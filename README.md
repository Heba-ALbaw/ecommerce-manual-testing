# E-commerce Website Manual Testing

## Project Overview
This project demonstrates manual testing of a demo e-commerce website.

The objective is to validate core e-commerce functionality such as:

- User login
- Product search
- Add to cart
- Checkout process
- Input validation

## Website Tested
https://demo.opencart.com/

## Testing Types Covered

- Functional Testing
- Negative Testing
- Boundary Testing
- Usability Testing

## Tools Used

- Browser (Chrome / Firefox)
- Excel (Test cases & bug reports)
- GitHub for documentation

## Project Structure

Test-Plans → Test plan documentation  
Test-Cases → Detailed manual test cases  
Bug-Reports → Logged defects  
Screenshots → Evidence of bugs  
Test-Reports → Execution summary
